## 关于本分叉

原始项目的介绍、宣传、演示与更新日志等信息已移除，只保留满足个人需求所需的部分。完整改动以提交历史为准。

面向 Cocos Creator 3.8+ 的 MCP 服务插件：作为编辑器扩展加载后，在面板里设置端口与访问令牌并启动，对外暴露本机 HTTP MCP 端点（默认 `http://127.0.0.1:3000/mcp`），供 MCP 客户端驱动编辑器（场景 / 节点 / 组件 / 预制体 / 资源 / 构建等）。

## 相对上游的主要改动（安全 / 协议）

- **反 DNS-rebinding**：校验 `Host` 头必须解析到 loopback。
- **来源白名单 + 可选 Bearer 鉴权**：新增 `authToken` 设置项与面板输入框；`allowedOrigins` 真正生效（不再无条件 `*`）。
- **MCP 协议修正**：通知消息（`notifications/*`）不再错误返回响应、新增 `ping`、`initialize` 回显客户端协议版本。
- **修复改端口重启竞态**：`stop()` 等待服务真正关闭再重启，避免 `EADDRINUSE`。
- **收敛 JSON “修复”逻辑**：移除会破坏合法内容的全局引号/换行改写，仅保留保守处理。
- **修复保存设置的 `enableDebugLog` 字段未生效问题**。
